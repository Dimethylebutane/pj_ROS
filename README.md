# pj_ROS
projet ROS2 M2 ENSAM


# Utilisation
## Épreuve du couloir
```> ros2 run pj_ROS couloir.launch.py```
## Épreuve de la ligne
```> ros2 run pj_ROS ligne.launch.py```
## Épreuve de la balle
```> ros2 run pj_ROS balle.launch.py```
## Trois pour le prix d'un
```> ros2 run pj_ROS total.launch.py```
